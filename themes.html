The JoltNotes Themes

The JoltNotes family has a few themes that you can choose from. You can change the theme by clicking the gear button in the top left corner.

The themes are:

JoltNotes - This is the default theme, it has a gray background and white text, and blue for the buttons.

Dark - This is a dark theme, it has a dark gray background and white text, and blue for the buttons.

Super Dark - This is a dark theme, it has a black background and white text, and white for the buttons.

Arctic White - This is a white theme, it has a white background and black text, and blue for the buttons.

Terminal Amber - This is a amber theme, it has a retro gray background and amber text, and amber for the buttons.

Terminal Green - This is a red theme, it has a retro gray background and green text, and green for the buttons.

Pretty Pink - This is a pink theme, it has a pink background and black text, and white for the buttons.


Advanced Info:

The themes are stored in the main file, JoltNotesLite.py. I use customtkinter for the UI, so every single theme looks like this:

def color_change_JoltNotes():
toolbar.configure(fg_color='#262626')
window.configure(fg_color='#171717')
textbox.configure(fg_color='#171717', text_color='white')
theme_label.configure(text_color='white')
bold_button.configure(text_color='white')
italic_button.configure(text_color='white')
underline_button.configure(text_color='white')
heading_button.configure(text_color='white')
heading2_button.configure(text_color='white')
heading3_button.configure(text_color='white')
strickethrough_button.configure(text_color='white')
menu_button.configure(text_color='white')
settings_menu_button.configure(text_color='white')
website_button.configure(text_color='white')
save_theme("JoltNotes")

If you would like to customize a theme to your likeing, follow this guide below:

1. Open the JoltNotesLite.py file in a text editor.

2. Find the theme you want to edit.

3. Change the colors to your liking.

4. Save the file.

5. Open JoltNotesLite.app and the theme should be changed.

This has not been tested, but in theory it should. If this method does not work I will update it at some point.
