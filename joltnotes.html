The JoltNotes Family

I'm currently working on JoltNotes. JoltNotes is a note-taking application that is written with Python and customtkinter using VS Code. I'm currently developing three different versions of the software.



JoltNotes(Coming soon) - This is desgined to be big and full of features, this will most likely be the hardest to develop.

JoltNotes Lite - This is a lightweight simple version of JoltNotes. With the full releases at some point, this will be missing features compared to JoltNotes.

JoltminiNotes - This was developed with sticky notes in mind, very simple in terms of everything.
